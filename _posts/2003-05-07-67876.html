All those classic "magic" number tricks are algebra :-)

Though strictly speaking, this one will only work for 2003, you can make it
work for any year if you just increment 1753/1752 each year. (It's not
coincidence that 2003 and 1753 both end in 3.)

> 1. First of all, pick the number of times a week that you would like to
> have dinner out. (try for more than once but less than 10)
>
> 2. Multiply this number by 2 (Just to be bold)
>
> 3. Add 5. (for Sunday)
>
> 4. Multiply it by 50 - I'll wait while you get the
> calculator................

((EatOut x 2) + 5) x 50

That's just a tricky way of multiplying the number of times you want to eat
out by 100. In other words, it will move the number of times you want to eat
out over to the hundreds position. So if you want to eat out 4 time a week,
you'll end up with 400 (plus some extra that gets subtracted off later).

So it's really:

((Eatout x 2) x 50) + 5 x 50) =
(Eatout x 100) + 250

> 5. If you have already had your birthday this year add 1753.... If you
> haven't, add 1752..........
>
> 6. Now subtract the four digit year that you were born.

All this does is subtract your birth year from 2003. Notice that if you add
250 (the extra number added to (Eatout x 100)) to 1753 you get ... 2003.

(Eatout x 100) + ((250 + 1753) - 1956)

or (Eatout x 100) + 2003-1956 :-)
