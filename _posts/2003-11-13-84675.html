Somebody on this list mentioned "CountDown" by Cadaco. I bought this
game at Target and really like it. It is simple simple simple - but
somehow a bit addicting. This is especially good for kids who are busy
enjoying learning all the basic "math facts."

The reason I like it so much is that it has a lot of scope for "playing
around" with the rules - make up your own varations.

I have some variations of this on my blog -
