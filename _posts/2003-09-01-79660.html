If you send a message to this group and get back a "bounced message notice," saying that the group doesn't exist, it most likely means you have left the "dash" in the group name. The name has been changed to UnschoolingDiscussion@yahoogroups.com WITHOUT a dash between Unschooling and Discussion.

Hitting "reply" to a message will not work if it is a message that was sent before the list name was changed and you'll need to go up to your "To" or "Send" box and manually take out the "dash" in the name.

Yahoogroups seems to be working again, it was very very uncooperative overnight, so I hope we don't lose much of the momentum of the discussion due to the name change. If you tried to send mail before and it bounced, please resend it now with the new name in the "Send to" box.
