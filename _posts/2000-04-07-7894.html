Good Morning all,

We have just set up a channel for the children to chat in, on irc.
The channel will not show up if you do a search for it, because it is
set as secret. The only people who will be able to access the
channel, must have the password.

The channel is set up on Efnet. Some of the irc servers which will
reach efnet are:
irc.mcs.net
irc.nethead.com
irc-w.frontiernet.net
irc.core.com
irc.concentric.net
irc.sprynet.com
irc.lightning.net
irc.telia.se
irc.prison.net
irc.mindspring.com
irc.idle.net
irc.total.net
irc.idirect.ca
irc.etsmtl.ca

There are many other servers. If anyone has difficulty, please
email me, and I will send you additional servers.

The channel name is #unschoolkids. To join the channel type
"/join #unschoolkids September" September is the password.

You can download mIRC at www.mirc.co.uk, look in the download
section for mIRC32. To set this software up, after you download it,
run the file, it will ask you where to install it and if you want to
create shortcuts, allow everything it wants. When it is done, start
mIRC, it will prompt you with a bunch of things to setup. There is
Full Name, E-Mail address, Nickname and Alternative nickname.
Fill everything in correctly. On the left panel, it will say Connect,
IRC, Sounds, Mouse, Dcc, Display and a bunch of other tabs.
Double click on Connect, then click on Identd. In the userid space,
enter unschool. mIRC is now setup and you can connect to a
server, once everything is setup, press Ok. Now type "/server <irc
server>" where irc server is one of the servers I have included
above. Once you are connected, you will see a bunch of junk flash
by your screen, you can ignore it. Now type in the channel name,
and the password.

The above paragraph is compliments of my not quite 13 year old
son. :)
