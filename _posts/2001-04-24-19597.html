the problem is the virus send the mail out like it is a personal message, not through the user group. If you are using outlook express, you can set up a filter to automatilcally remove the infected e-mail.
Click on tools and select message rules mail.
select new mail rue
select where the message body contains specific word
Select delete it for the action
in the rule description box, click on contains specific words and edit it to say "look to the attachment".
edit rule name 
click ok
 
after you do this, everything with the virus will automatically go straight to the deleted items folder
