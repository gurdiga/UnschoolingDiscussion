Someone asked about game programming books. Here is
an online computer game creation site that my dh & 8
yr old son have used to make their own fun games.

http://www.gamemaker.nl/

Game Maker
Did you always want to design computer games? But you
don't want to spend a lot of time learning how to
become a programmer? Then you came to the right place.
Game Maker is a program that allows you to make
exciting computer games without the need to write a
single line of code. Using easy to learn drag-and-drop
actions you can make professional looking games in
little time. Games with backgrounds, animated
graphics, music and sound effects. And once you get
more experienced there is a built-in programming
language that gives you full flexibility. What is
best, Game Maker can be used free of charge.

enjoy!

from my dh...

What Q and I did, was go to this web page:

http://www.cs.uu.nl/people/markov/gmaker/tutorial.html

And downloaded the "Your first game" tutorial. We both
sat down and I read the tutorial to him, and let him
"program the game". It's the simplest "game" you can
create ... two bouncing balls. no controls.

we then we added stuff to that game, like trying to
click the balls with the mouse, making the balls
bigger, adding barriers, etc.
